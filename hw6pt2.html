Ration

The proportion of college graduates with science or engineering degrees.

My original data focused on the number of science/engineering graduates per county. To create the ration for this assignment, I used the total number of college graduates per county. Dividing the two values provides the number of science/engineering majors in the overall number of graduates. This new cloropleth is a better representation of this data as it no longer simply highlights counties with higher populations.

Equal Count (Quartile) Classification

This classification divides the data so there are equal groups in each of the five categories. This makes the shading of the map even, and it makes it easy for the viewer to see each category distinctly. However, one group may contain a wide range of data. For example, in this map, the fifth category has a county with 30.5% of graduates in the same category as a county with around 45% of graduates, which may be confusing.

Natural Breaks Classification

The natural breaks classification groups similar values together. For this reason, values in a group do not tend to vary as far from one another, making the map more representative of the data and less misleading.

Pretty Breaks Classification

Using the pretty breaks classification rounds the values of the data set. This makes the legend of the map easy to understand and "pretty" to look at.
